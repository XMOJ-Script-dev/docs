# 自动提交当年代码

快速获得高正确率，一直是所有OIers的追求，无数人为了达成这个目标，废寝忘食，设计出了各种OI比赛中无法实现的方法（包括但不限于搜题解，提前准备板子）但是大家都知道，这是不可能在比赛中成功的。

我们，也曾被这个问题所困扰，但是，今天我们带来了一个算是完美的解决方案！

现在，当你点开一个比赛界面，你会发现，多了一个按钮“自动提交当年代码”。点击这个按钮，你曾经对过的题目如果出现在题单内就会被自动提交

**但是**这会给你带来“疑似抄袭当年代码”的黄色牌子，并在这个比赛内永久保留！所以，各位OIers们，真的~$\color{black}{\fcolorbox{black}{black}{不}}$~不要点！

~~快去试试吧！~~
