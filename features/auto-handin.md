# 自动提交当年代码

## 功能介绍

"自动提交当年代码"功能允许您在**竞赛/作业**中，一键将曾经 AC 过的题目代码重新提交，适用于每年需要重新提交当年代码的比赛场景。

## 使用方法

1. 点击右上角**用户名** → **插件设置**
2. 找到"**自动提交当年代码**"选项，点击开关将其**启用**
3. 前往**竞赛 & 作业**，找到目标比赛并进入
4. 题目列表上方将出现"**自动提交当年代码**"按钮
5. 点击按钮，脚本将自动遍历历史提交，找到曾经 AC 的代码并重新提交

## 注意事项

::: danger 谨慎使用
点击按钮后，系统会自动提交曾经 AC 的代码。由于此操作具有**明显的批量提交特征**，约有 **90%** 的概率被系统标记为 🟡 **疑似提交当年代码**，并在提交记录和排行榜上显示黄色警告标记。

**请在清楚了解后果的情况下再使用此功能。**
:::

- 由于题目数据、时间限制或内存限制可能已变更，重新提交不一定能 AC（可能出现 WA、TLE、MLE 等结果）
- XMOJ-Script **不对重新提交的正确性负责**
- 提交完成后请手动检查各题目的状态

## 常见问题

**Q：启用后为什么没有看到按钮？**

请确认您已进入比赛详情页（题目列表页），而非比赛列表页。同时确认插件设置中该功能已成功开启。

**Q：可以停止正在进行中的自动提交吗？**

目前版本暂不支持中途取消。建议在确认操作前仔细考虑。
