# 图床功能

某天，万能的原开发者@chenlangning 在遨游OI海洋的时候突发奇想“如果我们脚本的讨论区和某知名OJ一样有图床那该多好呀！”本着造福人类，客户至上的原则，TA行动了！！！

所以，我在此愉快地宣布，小明的OJ增强脚本讨论区图床功能，上线！

接下来，请允许我介绍一下参与此次发布会的领导。他们是：小明的OJ增强脚本开发组前组长 @chenlangning 和 开发组成员 @zhuchenrui2 。【鼓掌】

大家好，我是本次活动的主持人，蒟蒻PM @shanwenxiao 。【booo】

在金秋十月，我们欢聚云端，共同见证小明的OJ论坛的有一次更新！开发组团队带来了自制图床！

使用方式：

在任何一个讨论的发送框中右键粘贴或者在你臭名昭著的键盘上点击ctrl+v，就可以上传图片啦！

快去试试吧！（小贴士：由于服务器位置原因，上传较慢，请耐心等待。）
