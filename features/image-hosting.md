# 图床

XMOJ-Script 为讨论区和短消息提供了内置**图片上传与存储**功能，支持永久存储和代理访问。

## 使用方法

1. **复制**一张图片（截图、从网页复制均可）
2. 在任意**讨论区发帖框**或**短消息输入框**中，按 <kbd>Ctrl</kbd>+<kbd>V</kbd>（macOS 为 <kbd>⌘</kbd>+<kbd>V</kbd>）粘贴
3. 脚本自动将图片上传至服务器，并在输入框中插入图片链接
4. 发送帖子/消息后，其他用户即可正常查看图片

::: tip 提示
也可以在输入框中**右键 → 粘贴**来触发上传。
:::

## 注意事项

::: warning 使用限制
- 由于服务器地理位置关系，图片上传和加载速度可能较慢，请耐心等待
- 禁止上传违反[用户协议](../legal/eula)、公序良俗或另行通知禁止上传的内容
- 管理组有权删除不符合规定的图片
- 多次上传违规图片可能导致该功能被**封禁**
:::

## 上传失败怎么办？

如遇到以下情况，请参考对应解决方案：

| 错误表现 | 解决方案 |
|---------|---------|
| 粘贴后无反应 | 确认图片已正确复制；尝试截图后再粘贴 |
| 上传进度卡住 | 检查网络连接；关闭代理/VPN 后重试 |
| 显示服务器错误 | 联系管理员，或稍后重试 |
| 图片显示损坏 | 尝试重新上传；检查原图是否有效 |

如遇到不理解的错误信息，欢迎在[讨论区反馈版](https://www.xmoj-bbs.me)提问。
