# 功能总览

XMOJ-Script 是一个功能丰富的浏览器用户脚本，覆盖界面美化、竞赛辅助、数据获取、社区互动等多个方面。

## 界面增强

| 功能 | 说明 |
|------|------|
| Bootstrap 5 样式库 | 全面更新界面样式，更现代、更美观 |
| 暗色模式 | 减少眼部疲劳，支持自动跟随系统设置 |
| 界面动画 | 为各组件添加过渡动画，提升交互流畅感 |
| 中文翻译 | 统一翻译页面中残留的英文内容 |
| 重新排版 | 优化页面布局，提高可读性 |
| 状态单位显示 | 内存（KB/MB）和耗时（ms）自动添加单位 |
| 对错符号替换 | 将题目结果标识替换为直观的 ✓ / ✗ |
| 关键词替换 | 替换特定关键词为更易理解的表述 |
| 去除多余提示 | 清理界面上重复或多余的提示信息 |
| 链接按钮化 | 将方括号链接替换为样式统一的按钮 |
| 移除无用功能 | 隐藏平台上无法使用或已下线的功能入口 |

## 竞赛辅助

| 功能 | 说明 |
|------|------|
| ACM 排名查看与下载 | 在比赛页面查看 ACM 榜单并支持导出 |
| 自动刷新比赛列表与排名 | 定时自动刷新，实时获取最新信息 |
| 界面倒计时 | 在比赛列表页面显示剩余时间倒计时 |
| 一键打开所有题目 | 在比赛题目列表页面一键在新标签页打开全部题目 |
| [自动提交当年代码](./auto-handin) | 一键将历史 AC 代码重新提交到当年比赛 |
| 自动选择 O2 优化 | 提交代码时自动勾选 O2 编译优化 |
| 回放视频下载 | 在回放页面添加视频下载按钮 |

## 代码与提交

| 功能 | 说明 |
|------|------|
| [获取测试点数据](./get-data) | 查看错误运行的测试点输入/输出数据 |
| 代码比较 | 比较不同版本的提交代码 |
| 代码检查 | 提交前自动检测文件 IO 使用情况与编译错误 |
| 自动刷新状态页面 | 提交后状态页面结果自动实时刷新 |
| 复制测试样例 | 一键复制题面中的输入/输出样例 |
| 复制题目/题解内容 | 支持将题目或题解全文复制为文本或 Markdown |

## 用户与账号

| 功能 | 说明 |
|------|------|
| [用户评分系统](./rating) | 基于 AC 率的彩色评分，直观展示用户水平 |
| [徽章/Tag 系统](../community/badges) | 特定用户可获得专属徽章显示于用户名旁 |
| 自动跳转登录页 | 需要登录时自动重定向到登录界面 |
| 自动保存用户名密码 | 自动填充已保存的登录凭据 |

## 社区功能

| 功能 | 说明 |
|------|------|
| 讨论区 | 独立搭建的讨论服务，可发表评论和帖子 |
| 短消息 | 独立搭建的短消息服务，支持用户间私信 |
| [图床](./image-hosting) | 在讨论区和短消息中粘贴即可上传图片 |

## 其他功能

| 功能 | 说明 |
|------|------|
| 下载页面增强 | 在下载页面提供更多可用资源链接 |
| 修复登录跳转 | 解决登录后页面跳转异常的问题 |
| 学术模式 | 体验极致的学习环境（部分功能开发中）|
