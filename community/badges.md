# 徽章 / Tag

![徽章示例](/badge.png)

## 什么是徽章？

徽章（Badge / Tag）是显示在用户名旁的**特殊标识**，用于标记对社区有特殊贡献或特殊身份的用户。

## 如何获得徽章？

目前，通过以下方式可以获得徽章：

- **邀请新用户**：邀请其他用户安装并使用 XMOJ-Script
- **赞助项目**：通过 [Open Collective](https://opencollective.com/xmoj-script-dev) 赞助 XMOJ-Script 开发
- **贡献代码**：向 [XMOJ-Script 仓库](https://github.com/XMOJ-Script-dev/XMOJ-Script)提交有效的 Pull Request

如果您认为自己符合获得徽章的条件，请联系管理员。

## 如何编辑/展示徽章？

获得徽章后，您可以前往以下页面管理您的徽章展示：

👉 [https://www.xmoj.tech/modifypage.php](https://www.xmoj.tech/modifypage.php)

在该页面中，您可以选择显示哪些徽章、调整顺序等。

## 徽章显示位置

徽章将显示在以下位置：

- 讨论区帖子的作者信息旁
- 用户个人页面的用户名旁
- 短消息中的发送者信息旁
